# Sistema de Gestión Hotelera
Fecha: 11/02/2025
Autor: Andres Hincapie Ruiz
Este proyecto es el resultado de una prueba técnica para desarrollador Front-end, implementando una aplicación web de gestión hotelera utilizando React.

## Descripción

La aplicación permite la administración de hoteles locales y reservas, con dos roles principales:

### Agente de Viajes
- Crear y gestionar hoteles
- Administrar habitaciones disponibles
- Modificar precios y datos de hoteles/habitaciones
- Habilitar/deshabilitar hoteles y habitaciones
- Ver listado y detalles de reservas

### Viajero
- Buscar hoteles por fecha, capacidad y ciudad
- Seleccionar y reservar habitaciones
- Gestionar datos de huéspedes y contactos de emergencia
- Recibir confirmaciones por email

## Características Técnicas

- Framework: React 19
- Gestión de Estado: Context API
- Enrutamiento: React Router
- Estilos: CSS Modules
- Iconos: Font Awesome
- Datos: Mock Data simulada
